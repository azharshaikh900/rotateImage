rotateImage.html creates a dummy image of Blue colour and rotates it and displays both original and rotated image on the canvas.
It doesn't use any canvas APIs

By default rotation is done at an angle of 90 degrees. To change the angle parameter update angle variable in loadCanvas function.
By default the dimenstion of the image is 100x100 pixels. To change the dimesion of the image update the imageData variable in loadCanvas function.

Performance of the API can be seen in cosole of the browser. Press F12 to view the cosole in Google chrome.
The code rotates the 2K resolution image in under 2 seconds.
